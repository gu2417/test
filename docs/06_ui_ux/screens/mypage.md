# 마이페이지

Gmail 계정 페이지의 **큰 아바타 + 메타 + 통계 카드** 레이아웃.
한 페이지에 profile / security / stats / actions 4 영역을 여백으로 구분.

## 와이어프레임 (100×30)

```
╭──────────────────────────────────────────────────────────────────────────────────────╮
│ ←  마이페이지                                                                        │
╰──────────────────────────────────────────────────────────────────────────────────────╯


                        ╭──────╮
                        │  홍  │        홍길동
                        │      │        ● online · 오늘도 화이팅
                        ╰──────╯        hong123


                 ─────────────────  ·  ─────────────────


      활동                                보안

      1,243    메시지                    마지막 접속   방금 전
          5    참여한 방                 가입일        2026. 1. 15
         12    친구                      비밀번호      ••••••


                 ─────────────────  ·  ─────────────────


      [  프로필 수정  ]   [  비밀번호 변경  ]   [  설정  ]   [  로그아웃  ]

```

## 구성 요소

| 영역 | 컴포넌트 |
|------|----------|
| Hero | 큰 아바타(`4×3`) + 닉(title) + StatusChip + 상태메시지 + ID(`meta`) |
| 활동 통계 | 숫자(bold, title) + 라벨(caption) 3열 |
| 보안 | 메타 테이블: 마지막 접속 · 가입일 · PW 마스킹 |
| 액션 | Chip 버튼 4개. 가로 정렬, 포커스 시 inverse |

## 인라인 편집 (프로필 수정)

`[프로필 수정]` 선택 시 Hero 영역이 편집 모드로 전환(모달 없음):

```
                        ╭──────╮
                        │  홍  │        닉네임    [ 홍길동              ]
                        │      │        상태 메시지 [ 오늘도 화이팅     ]
                        ╰──────╯
                                        [ 취소 ]   [ 저장 ]
```

- `Enter` 저장 → `PROFILE_UPDATE` + 성공 시 Toast `✓ 저장되었습니다`.
- `Esc` 취소.

## 비밀번호 변경 (Modal)

클릭 시 `components.md#13-modal` 폭 40 모달:
```
╭─ 비밀번호 변경 ────────────────╮
│                                │
│  현재 비밀번호  [         ]    │
│  새 비밀번호    [         ]    │
│  새 비밀번호 확인[        ]    │
│                                │
│        [ 취소 ]   [ 변경 ]     │
╰────────────────────────────────╯
```

## 데이터 소스

- 진입 시 `MYPAGE` 1회 요청. 응답으로 Hero + 통계 + 보안 영역 채움.
- `마지막 접속` 은 `last_seen` 기반 상대 시간:

| 차이 | 표시 |
|------|------|
| <1분 | `방금 전` |
| <60분 | `N분 전` |
| <24시간 | `N시간 전` |
| 그 외 | `YYYY. M. D` |

## 키

| 키 | 동작 |
|----|------|
| `Tab` / `↓` | 액션 칩 순환 |
| `Enter` | 칩 실행 |
| `Esc` / `0` | MAIN 복귀 |
| `s` | 설정 바로가기 |
