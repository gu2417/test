# 메인 화면

Google Chat 의 **사이드바 + 메인 패널** 2-pane 레이아웃.
좌측은 네비게이션(내 프로필 + 탭 트리), 우측은 선택한 탭의 리스트.

---

## 1. 와이어프레임 (100×30 기준)

```
╭────────────────────────╮╭──────────────────────────────────────────────────────────╮
│ ◆ C Chat               ││ 친구                                               ⌕  ⋯ │
│                        ││                                                          │
│ ╭──╮                   │├──────────────────────────────────────────────────────────┤
│ │홍│  홍길동           ││  ONLINE · 3                                              │
│ ╰──╯  ● 오늘도 화이팅  ││                                                          │
│                        ││   ╭──╮                                                   │
├────────────────────────┤│   │김│  김철수                            · 방금        │
│                        ││   ╰──╯  점심 뭐 먹지                                     │
│  ▾  친구          12   ││   ● online                                               │
│  ▸  채팅           5 ❶ ││                                                          │
│  ▸  오픈채팅      24   ││   ╭──╮                                                   │
│  ▸  마이페이지         ││   │이│  이영희                            · 10분        │
│                        ││   ╰──╯  공부중                                           │
├────────────────────────┤│   ● busy                                                 │
│                        ││                                                          │
│  +  새 채팅            ││  ─────────────  ·  ─────────────                         │
│  ⚙  설정               ││  OFFLINE · 2                                             │
│                        ││                                                          │
│                        ││   ╭──╮                                                   │
│                        ││   │박│  박민준                            2시간 전      │
│                        ││   ╰──╯  (상태 메시지 없음)                               │
│                        ││   ○ offline                                              │
│                        ││                                                          │
╰────────────────────────╯╰──────────────────────────────────────────────────────────╯
╭──────────────────────────────────────────────────────────────────────────────────────╮
│ ▌ 명령 입력                                                  /help         ⏎ 전송    │

```

- 사이드바 폭: 26 고정 (터미널 < 90 이면 접혀서 상단 탭 바만 표시).
- 메인 패널 상단은 TopBar (타이틀 + 우측 액션).
- 하단 Composer 는 **전역 1개**, 화면과 무관하게 상시 노출.

---

## 2. 사이드바 구성

1. **Identity 카드** — 내 아바타 + 닉 + StatusDot + 상태 메시지 (1줄)
2. **네비게이션 트리** — 친구 / 채팅 / 오픈채팅 / 마이페이지
   - 각 항목 뒤에 카운트 뱃지 (미읽음 수 `❶`)
   - 선택 시 좌측 `▌` 액센트 + bold
3. **보조 액션** — `+ 새 채팅`, `⚙ 설정`, `▤ 관리자 패널` (admin 전용)

---

## 3. 탭별 메인 패널

### 3-1. 친구 탭 (기본)

섹션 순서: `받은 요청` → `ONLINE` → `BUSY` → `DND` → `OFFLINE`

**받은 요청 섹션** (미처리 요청 있을 때만):
```
   받은 요청 · 2

   ╭──────────────────────────────────────────────────────────────╮
   │ ╭──╮                                                         │
   │ │홍│  홍길동   hong123                     [수락]  [거절]   │
   │ ╰──╯  오늘도 화이팅                                          │
   ╰──────────────────────────────────────────────────────────────╯
```

**온라인 섹션:**
- 각 항목은 Card(아바타 + 닉 + 상태 메시지 + 우측 최근 활동 시간).
- Hover 시 우측에 `[DM]` `[프로필]` 퀵액션 칩.

**오프라인 항목:** `마지막 접속: 2시간 전` 을 상태 메시지 대신 표시.

**빈 상태:**
```
                     아직 친구가 없습니다.
                     /friend add <ID> 로 첫 친구를 추가해보세요.
```

### 3-2. 채팅 탭

그룹방 + 최근 DM 혼합 (최근 메시지 시간 내림차순).

```
   ╭──╮
   │컴│  # 컴공 스터디그룹             · 5분    ❶
   ╰──╯  홍길동: 과제 다들 했어?
   ● 12명

   ╭──╮
   │김│  ✉ 김철수                      · 방금   ❶
   ╰──╯  ㄴㄴ 지금 시작...
   ● online
```

- 방 이름 앞에 `#` (그룹) / `#` (오픈) / `✉` (DM) glyph.
- 오픈방 아바타: 방 이름 첫 글자 + `[오픈]` 칩.

**빈 상태:**
```
                     참여 중인 채팅이 없습니다.
                     /room create <이름> 또는 /join <room_id> 로 시작하세요.
```

### 3-3. 오픈채팅 탭

오픈채팅 탐색 패널. 상세는 [`screens/open_chat.md`](./open_chat.md) 참고.

- 상단에 검색 입력 `⌕ 검색어 입력...`
- 각 항목: RoomCard(`components.md#20`)
- 비번 있으면 `◈` 아이콘, 만원이면 `[FULL]`.

### 3-4. 마이페이지 탭

선택 즉시 [`screens/mypage.md`](./mypage.md) 화면으로 전환.
(패널 안 인라인 렌더 아님)

## 4. 그룹채팅방 생성 모달

`n` 키 또는 `+ 새 채팅` → 유형 선택:

```
╭─ 새 채팅 ───────────────────────────────────────╮
│                                                  │
│   어떤 채팅을 시작하시겠어요?                   │
│                                                  │
│   1. 1:1 DM          친구와 개인 대화            │
│   2. 그룹 채팅방     초대 기반 비공개 방         │
│   3. 오픈채팅방      누구나 참여 가능한 방       │
│                                                  │
│                        [ 취소 ]                  │
╰───────────────────────────────────────────────────╯
```

**1 선택 (DM):**
```
╭─ DM 시작 ───────────────────────────────────────╮
│                                                  │
│   상대방 ID 또는 닉네임                          │
│   ┌──────────────────────────────────────────┐  │
│   │ ▌                                        │  │
│   └──────────────────────────────────────────┘  │
│                                                  │
│   — 또는 친구 목록에서 선택 —                   │
│                                                  │
│   ╭──╮  김철수   ● online                        │
│   ╭──╮  이영희   ○ offline                       │
│                                                  │
│              [ 취소 ]   [ DM 시작 ]              │
╰───────────────────────────────────────────────────╯
```

**2 선택 (그룹채팅방):**
```
╭─ 그룹 채팅방 만들기 ────────────────────────────╮
│                                                  │
│   방 이름 *   ┌──────────────────────────────┐  │
│               │ ▌                            │  │
│               └──────────────────────────────┘  │
│               최대 30자                          │
│                                                  │
│   주제        ┌──────────────────────────────┐  │
│               │                              │  │
│               └──────────────────────────────┘  │
│               선택 · 최대 100자                  │
│                                                  │
│   최대 인원   ┌──────┐  기본 30                  │
│               │  30  │                          │
│               └──────┘                          │
│                                                  │
│   비밀번호    ┌──────────────────────────────┐  │
│               │  (없으면 공개)               │  │
│               └──────────────────────────────┘  │
│                                                  │
│              [ 취소 ]   [ 만들기 ]               │
╰───────────────────────────────────────────────────╯
```

성공 → CHAT 화면 진입 (새 방).

## 5. 유저 검색 결과 (`/find <keyword>`)

메인 패널이 검색 결과 패널로 임시 전환:

```
   ⌕  검색: "홍"                                                           ×

   검색 결과  3

   ╭──╮
   │홍│  홍길동   hong123
   ╰──╯  오늘도 화이팅                        [친구 추가]  [DM]
   ○ offline

   ╭──╮
   │홍│  홍영희   hong_yh
   ╰──╯  개발 중...                           [친구 추가]  [DM]
   ● online

   ╭──╮
   │홍│  홍민수   hongmin
   ╰──╯                                       [이미 친구]
   ● busy
```

- `[이미 친구]` = dim + 비활성.
- `[친구 추가]` 선택 → FRIEND_ADD_REQ → 결과 행에서 칩이 `[요청 전송됨]` 으로 교체.
- `Esc` 또는 `×` → 이전 탭 패널 복귀.

## 6. 키 바인딩

| 키 | 동작 |
|----|------|
| `1` `2` `3` `4` | 사이드바 탭 이동 (친구/채팅/오픈채팅/마이페이지) |
| `Tab` | 사이드바 ↔ 메인 패널 포커스 전환 |
| `↑` `↓` | 리스트 항목 이동 |
| `Enter` | 항목 열기 (DM 시작 / 방 진입 / 프로필) |
| `/` | Composer 로 포커스 + 슬래시 모드 |
| `n` | 새 채팅 모달 |
| `f` | 검색 입력 포커스 (오픈채팅 탭) |
| `g s` | 설정 (2키 단축키) |

## 7. 사용 가능한 커맨드

메인 화면 전용:
`/friend *`, `/dm <id>`, `/room create`, `/open create`, `/join [pw]`, `/rooms`, `/room search`, `/find`, `/status`, `/dnd`, `/mypage`, `/settings`, `/admin`, `/quit`

(전체 표: [`../commands.md`](../commands.md))

## 8. 알림 배너

- 화면 최상단(TopBar 위) 에 overlay 로 1행 삽입. 최대 3개 스택.
- 친구 요청 배너: 인라인 액션 `[1 수락] [2 거절]` 숫자 키 바인딩.
- Banner 컴포넌트(`../components.md#8`) 사용.

## 9. 좁은 터미널 (< 90열) 대응

사이드바 숨김 → 상단에 얇은 탭 바:

```
╭──────────────────────────────────────╮
│  친구   채팅 ❶   오픈   마이          │
╰──────────────────────────────────────╯
```

- 활성 탭은 밑줄 + bold.
- `Tab` 키 또는 `1..4` 로 이동.
