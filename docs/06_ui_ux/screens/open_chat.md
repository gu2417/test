# 오픈채팅 화면

누구나 발견·참여할 수 있는 오픈채팅방 탐색·참여 전용 화면.
메인 사이드바에서 `오픈채팅` 탭 또는 `/rooms open` 으로 진입.

---

### 방 카드 구성

```
╭──────────────────────────────────────────────────╮
│ #  방이름 (굵음)                   cur/max   [◈] │   ← ◈ = 비밀번호 방
│    주제 한 줄 (fg.secondary, 최대 50자 truncate) │
╰──────────────────────────────────────────────────╯
```

- 비밀번호 방: 우측 `◈` 아이콘 + `fg.warn`.
- 만원(cur == max): `cur/max` 텍스트가 `feedback.error` + `[FULL]` 칩.
- 내가 이미 참여 중인 방: 우측에 `● 참여중` chip.

## 2. 실시간 검색

검색 입력 시 `ROOM_SEARCH` 패킷 전송. 디바운스 **300ms**.

```
   ┌──────────────────────────────────────────────────┐
   │ ⌕  스터디 ▌                                      │
   └──────────────────────────────────────────────────┘

   검색 결과  2

   ╭──────────────────────────────────────────────────╮
   │ #  개발 스터디                           8/30   │
   │    알고리즘 · 코딩테스트 풀이 매일              │
   ╰──────────────────────────────────────────────────╯

   ╭──────────────────────────────────────────────────╮
   │ #  컴공 스터디그룹                      12/30  ◈ │
   │    운영체제 / 네트워크 / DB 함께 공부           │
   ╰──────────────────────────────────────────────────╯
```

검색어를 지우면 전체 목록으로 복귀.

## 3. 방 상세 → 참여 플로우

방 카드에서 `Enter`:

### 3-1. 공개방 참여 확인

```
╭─ 방 참여 ───────────────────────────────────────╮
│                                                  │
│   # 개발 스터디                                  │
│   알고리즘 · 코딩테스트 풀이 매일               │
│   멤버  8 / 30                                   │
│                                                  │
│   오픈채팅 닉네임 (선택)                         │
│   ┌──────────────────────────────────────────┐   │
│   │  (기본 닉네임 사용)                      │   │
│   └──────────────────────────────────────────┘   │
│   비워두면 기본 닉네임 "홍길동" 으로 참여합니다. │
│                                                   │
│                   [ 취소 ]   [ 참여하기 ]         │
╰───────────────────────────────────────────────────╯
```

### 3-2. 비밀번호 방 참여

```
╭─ 비밀번호 입력 ─────────────────────────────────╮
│                                                  │
│   # 컴공 스터디그룹                              │
│   이 방은 비밀번호로 보호되어 있습니다.         │
│                                                  │
│   비밀번호  ┌───────────────────────────────┐   │
│             │  ••••••                       │   │
│             └───────────────────────────────┘   │
│                                                  │
│   ✗ 비밀번호가 맞지 않습니다.   ← 실패 시 표시  │
│                                                  │
│                   [ 취소 ]   [ 확인 ]            │
╰───────────────────────────────────────────────────╯
```

### 3-3. 만원 방

```
╭─ 참여 불가 ─────────────────────────────────────╮
│                                                  │
│   # 점심 뭐 먹을까 (50/50 FULL)                 │
│   최대 인원에 도달했습니다.                     │
│                                                  │
│                        [ 확인 ]                  │
╰───────────────────────────────────────────────────╯
```

참여 성공 → CHAT 화면으로 전환 + 입장 시스템 메시지 렌더.

## 4. 오픈채팅방 생성 (+ 버튼 / `/open create`)

```
╭─ 오픈채팅방 만들기 ─────────────────────────────╮
│                                                  │
│   방 이름 *   ┌──────────────────────────────┐  │
│               │ ▌                            │  │
│               └──────────────────────────────┘  │
│               최대 30자                          │
│                                                  │
│   주제        ┌──────────────────────────────┐  │
│               │                              │  │
│               └──────────────────────────────┘  │
│               최대 100자 · 선택                  │
│                                                  │
│   최대 인원   ┌──────┐  기본 30, 최대 500       │
│               │  30  │                          │
│               └──────┘                          │
│                                                  │
│   비밀번호    ┌──────────────────────────────┐  │
│               │  (없으면 공개)               │  │
│               └──────────────────────────────┘  │
│               선택 · 비우면 공개방              │
│                                                  │
│                   [ 취소 ]   [ 만들기 ]          │
╰───────────────────────────────────────────────────╯
```

성공 시 `ROOM_CREATE_RES` 수신 → 생성된 방 CHAT 화면으로 바로 진입.

## 5. 오픈채팅 닉네임 변경 (채팅 중)

채팅 화면에서 `/open_nick <새닉네임>` 또는 `⋯` 메뉴 → `닉네임 변경`:

```
╭─ 오픈채팅 닉네임 변경 ──────────────────────────╮
│                                                  │
│   이 방에서만 사용할 닉네임을 입력하세요.       │
│                                                  │
│   ┌──────────────────────────────────────────┐  │
│   │ ▌ 홍길동                                 │  │
│   └──────────────────────────────────────────┘  │
│   비워두면 기본 닉네임 사용                      │
│                                                  │
│                   [ 취소 ]   [ 변경 ]            │
╰───────────────────────────────────────────────────╯
```

성공 시 Toast `✓ 닉네임이 변경되었습니다`. 채팅 화면 TopBar 닉 업데이트.

## 6. 키 바인딩

| 키 | 동작 |
|----|------|
| `↑` `↓` | 방 카드 이동 |
| `Enter` | 참여 확인 모달 열기 |
| `/` | Composer 포커스 |
| `n` | 오픈채팅방 만들기 모달 |
| `f` | 검색 입력 포커스 |
| `Esc` | 검색어 초기화 / 모달 닫기 |
| `Tab` | 사이드바 ↔ 패널 포커스 전환 |
